# Project Max Finder 
 
## Описание
Проект для нахождения максимального значения в списке чисел с возможностью интерактивного ввода данных.

## Функциональность
 1. Поиск максимального значения в списке целых чисел 
 2. Интерактивный ввод данных через консоль 
 3. Валидация вводимых данных 
 4. Вывод результатов в удобном формате

  ## Использование

  ### Запуск программы
  \`\`\`bash
  javac src/Main.java
  java -cp src Main
