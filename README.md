# Project Max Finder 
 
## Описание
Проект для нахождения максимального значения в списке чисел с возможностью интерактивного ввода данных.

## Функциональность
     Поиск максимального значения в списке целых чисел 
     Вывод результатов в удобном формате

  ## Использование

  ### Запуск программы
  \`\`\`bash
  javac src/Main.java
  java -cp src Main
